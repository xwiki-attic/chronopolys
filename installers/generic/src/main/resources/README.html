CHRONOPOLYS Overview

CHRONOPOLYS is a second generation groupware, features professional features like phases, tasks, meetings, wiki, file sharing, comments, user rights, LDAP authentication, delays and quality overview.

CHRONOPOLYS is also based on XWiki engine that provides an advanced form and scripting engine which makes it an ideal development environment for constructing data-based intranet applications. It has powerful extensibility features, supports scripting, plugins and is based on a highly modular architecture. The scripting engine allows to access a powerful API for accessing the XWiki repository in read and write mode.

CHRONOPOLYS is used by major companies around the world and AVANE provides support and services for professional usage of CHRONOPOLYS.

CHRONOPOLYS is also provided as a hosting service on http://www.avane.com/
